Codabra Icon

To save this icon for your VSCode extension:

  1. Right-click on the icon above and select "Save image as..."
  2. Save it as "icon.png" in your extension's root directory
  3. If your browser doesn't support saving SVG as PNG directly, you can:
    • Take a screenshot of the icon (ensure it's 128x128 pixels)
    • Use an online SVG to PNG converter
    • Open this HTML file in a browser, then use a screen capture tool
  4. Verify that the icon.png file is in the root directory of your extension
